For Students:

Using the Program

  • There are ten scenes to be filled in
  • Each scene has two resizable text boxes
  • The characters are divided up into two groups: Constructive and Destructive
  • Fill out each text box depending on which group the character is in
  • Dialogue: Enter what the person is saying using either Distressed System or Effective System Concepts
  • Concept: What concept are you using?
  • Reason: Why are you using this concept?
  • Once all ten scenes are completed, a CSV file (with all of your responses) will automatically download
  • Click RUN PROGRAM button above when ready to begin

For All:


At this point in time, this program is just a passive input of data. More interactivity will be added as time goes on.

Purpose

  • Create a conflict script
  • Provide users with a CSV output of the script
  • Give students a different and interactive outlet for understanding conflict resolution concepts

Goals

  • Reflect on concepts explored in CUNY John Jay SOC206
  • One character uses Distrssed System concepts
  • Other character uses Effective System concepts
